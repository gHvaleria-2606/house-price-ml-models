# house-price-ml-models
This project was completed as the final project for the course CSE 2107 at Washington University in St. Louis. This notebook was adapted for clearer presentation and improved readability.

**Collaborators:** Valeria Garcia Hernandez, An Nguyen

## Overview
This project builds and compares multiple machine learning models to predict house prices using the Melbourne Housing Snapshot dataset.

The goal is to evaluate how different regression models perform and to understand how feature engineering and data preprocessing impact model accuracy.

## Models Implemented
- **Linear Regression**
- **K-Nearest Neighbors (KNN)**
- **Random Forest**

## Key Steps
- Conducted **exploratory data analysis (EDA)** to understand data distributions and relationships
- Performed data cleaning and **preprocessing**, including handling missing values and encoding categorical variables
- Applied **feature engineering** to improve model performance
- Used **feature seletion** techniques to reduce noise and improve interpretability
- Trained and evaluated **regression models**
